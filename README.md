
# Mini MLOps Pipeline Design for Online Transportation Fare Estimation Based on Trip and Weather Data

This repository contains a Data Mining MLOps mini project for a learning simulation of online transportation fare estimation.

## Short Description

The project builds a simple MLOps workflow from data understanding to final presentation packaging. The model estimates `price` from trip features such as distance, cab type, source, destination, service name, and time-based features.

This is a learning simulation only. It is not a real production pricing system.

## Folder Structure

```text
api/             Local FastAPI skeleton and request example
data/raw/        Original raw datasets, not modified
data/processed/  Cleaned dataset generated after preprocessing
diagrams/        MLOps pipeline diagrams
models/          Saved baseline model and model registry
notebooks/       Step-by-step project notebooks
reports/         Audit, model, checklist, CD, and final reports
slides/          Presentation outline
src/             Reusable preprocessing, training, validation, and evaluation code
```

## Dataset Files

- `data/raw/cab_rides.csv`
- `data/raw/weather.csv`
- `data/processed/cleaned_cab_rides.csv`

Raw cab ride data has 693,071 rows. Cleaned cab ride data has 637,976 rows. Weather data is not merged yet because time and location alignment must be planned carefully.

## Dataset Download Instructions

The dataset is not included in this GitHub repository because the files are large.

Dataset source:
https://www.kaggle.com/datasets/ravi72munde/uber-lyft-cab-prices

After cloning this repository, download the dataset manually from Kaggle and place the files in this structure:

```text
data/raw/cab_rides.csv
data/raw/weather.csv
```

The notebooks and scripts expect the raw dataset files to exist in the `data/raw/` folder.

The cleaned dataset can be generated by running the preprocessing notebook:

```text
notebooks/02_preprocessing_baseline_model.ipynb
```

After preprocessing, the generated cleaned file should be located at:

```text
data/processed/cleaned_cab_rides.csv
```

This project is a learning simulation for a Data Mining MLOps mini project. It is not a real production pricing system.

## How to Run Notebooks

Open the notebooks in order and run each one from top to bottom:

1. `notebooks/01_data_understanding.ipynb`
2. `notebooks/02_preprocessing_baseline_model.ipynb`
3. `notebooks/03_model_evaluation_quality_checklist.ipynb`
4. `notebooks/04_cd_scenario_pipeline_design.ipynb`
5. `notebooks/05_final_summary_presentation_preparation.ipynb`

## How to Run Local API

Install dependencies:

```bash
pip install fastapi uvicorn joblib pandas scikit-learn
```

Run from the project root:

```bash
uvicorn api.app:app --reload
```

Example request body is available in `api/request_example.json`.
